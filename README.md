## 项目难点

1. 强大的layout组件,包含主题、布局、工具栏等相关信息
2. 字体以及图标的标准解决方案
3. 错误日志收集打印
4. 国际化
5. 代码文件自动生成plop.js
6. 基于文件系统的路由
